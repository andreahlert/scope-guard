# Changelog

All notable changes to Scope Guard will be documented in this file.

## [0.0.1] - 2025-02-05

### Added
- Initial release
- `UserPromptSubmit` hook to capture original prompt
- `PostToolUse` hook to track file changes (async)
- `Stop` hook with Haiku agent for scope evaluation
- Session isolation for parallel instances
- Git diff integration for change analysis
- `cleanup-sessions.js` script for maintenance
- Zero dependencies, zero config
